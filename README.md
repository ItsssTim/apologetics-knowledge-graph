# Apologetics Knowledge Graph

A structured map of contemporary evangelical apologetics: argument trees
synthesised from popular talks and debates, with objections, responses,
tactical patterns, and the gaps each case leaves open.

Start at `index.html`, or use `browse.html` to search every tree by topic,
expert, objection, or Q-ID.

This is a static site (HTML/CSS/JS only). The search runs entirely in the
browser, so there is no backend and no data is collected from visitors.
